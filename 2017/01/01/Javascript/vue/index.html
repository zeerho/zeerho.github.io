计算属性

  • 计算属性的值基于它的依赖进行缓存，只在必要时才重新执行函数。
  • 当函数中用到的某个属性变化时，计算属性的值会自动更新。
  • 计算属性可以跟普通属性一样使用。
  • 计算属性只有真正用于应用中时才会计算。
